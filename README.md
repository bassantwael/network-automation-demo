# Network Automation Demo

Simple network automation demo that reads device definitions from YAML and prints them.
This project is a learning / demo piece showing:
- Python scripting (yaml parsing)
- Config management (YAML)
- Git branching workflow (feature branches + PRs)
- CI via GitHub Actions

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/automation.py
